1.2D转换
  1.倾斜
    1.什么是倾斜
      改变元素在页面中的形状
    2.语法
      属性：transform
      取值：
          1.skewX(xdeg)让元素向着x轴方向产生倾斜效果，实际上改变的是y轴的角度值
	     x取值为正,y轴逆时针倾斜
	     x取值为负,y轴顺时针倾斜
	  2.skewY(ydeg)让元素向着y轴产生倾斜效果，实际上改变就是x轴的角度值
	     y取值为正,x轴顺时针倾斜
	     y取值为负,x轴逆时针倾斜
	  3.skew(xdeg) 等同于skewX()
	  4.skew(xdeg,ydeg)
	 练习：
	   1.页面中创建一个200*200的元素，设置背景色
	   2.鼠标悬停时，该元素向右偏移200px,向下偏移200px,旋转135deg,向x轴倾斜45deg,同时放大1.5倍。
2.3D转换
  1.透视距离
    模拟人的眼睛到3D转换元素之间的距离（z轴）
    属性：perspective
    注意：该属性要加在3D转换元素的父元素上
  2.3D旋转
    属性：transform
    取值：
        1.rotateX(xdeg) 
	  以x轴为中心轴，旋转元素的角度
	2.rotateY(ydeg)
	  以y轴为中心轴，旋转元素的角度
	3.rotateZ(zdeg)
	  以z轴为中心轴，旋转元素的角度
        4.rotate3D(x,y,z,ndeg)
	  x,y,z取值为大于0的数字，表示该轴要参与旋转，取值为0则不参与旋转
	  rotate3d(1,0,0,45deg)//只有x轴旋转45度
	  rotate3d(1,1,1,10deg)
3.过渡
  1.什么是过渡
  使得CSS属性值，在一段时间内平缓变化的效果
  2.过渡属性
    1.指定过渡属性
      属性：transition-property
      作用：指定哪个属性值在变化的时候需要使用过渡效果来体现。
      取值：
          1.all 能够使用过渡属性，一律用过渡体现
	  2.具体属性名
	    transition-property:background;
	    当前背景的属性在发生改变时用过渡给体现出来
	    transition-property:border-radius;
	    当前倒角的属性在发生改变时用过渡给体现出来
	    transition-property:all;
            当所有能够使用过渡的属性在发生改变时用过渡给体现出现（常用）

	  能够使用过渡的属性：
	    1.颜色属性
	    2.取值为数字的属性
	    3.转换属性
	    4.阴影
	    5.渐变属性
	    6.visibility属性
    2.指定过渡时长
      作用：指定在多长时间内完成过渡操作
      属性：transition-duration
      取值：以s或ms为单位的数字
            1s=1000ms
    3.指定过渡速度时间曲线函数
      属性：transition-timing-function
      取值：
        1.ease 慢速开始，快速变快，慢速结束
	2.linear 匀速
	3.ease-in 慢速开始，加速结束
	4.ease-out 快速开始，慢速结束
	5.ease-in-out 慢速开始和结束，中间先加速再减速
    4.指定过渡延迟时间
      属性：transition-delay
      取值：以s和ms为单位
  3.过渡属性的编写位置
    1.将过渡放在元素声明的样式中
      既管取，又管回
    2.将过渡放在触发的操作中（hover）
      只管取，不管回
  4.过度的简写
    transition:property duration timing-function delay;
    ex：
    transition:all 2s;

    练习：
      1.创建一个100*100的元素，将其设置为一个圆形填充背景色
      2.鼠标悬停时，5秒钟之内，向右偏移500px的同时再旋转3圈
      3.鼠标移出去时，元素还能再移回来
4.动画
  1.什么是动画
    使用元素从一种样式逐渐变为另一种样式，即将多个过渡效果放到一起。
    动画的通过"关键帧"，来控制动画的每一步
       关键帧：
         1.动画执行的时间点
	 2.在该时间点上的样式是什么
  2.动画的使用步骤
    1.声明动画
      为动画起名
      定义关键帧
      @keyframes 动画名称{
        /*定义该动画中所有的关键帧*/
         0%{
	    /*动画开始时元素的样式*/ 
	 }
	 50%{
	    /*动画执行到一半时元素的样式*/
	 }
	 100%{
	    /*动画结束时元素的样式*/
	 }
      }
      做兼容性处理加前缀：-o-,-moz-,-ms-,-webkit-

       @-webkit-keyframes 动画名称{
        /*定义该动画中所有的关键帧*/
         0%{
	    /*动画开始时元素的样式*/ 
	 }
	 50%{
	    /*动画执行到一半时元素的样式*/
	 }
	 100%{
	    /*动画结束时元素的样式*/
	 }
      }

     
     练习：
       声明一个动画，要求背景颜色和倒角效果在以下几个关键帧上做变化。
          0%   red 0%
	  25%  yellow 50%
	  50%  blue 0%
	  75%  pink 50%
	  100% green 0%


    2.为元素调用动画
      指定元素调用的动画名称以及各个参数属性
      1.animation-name
        作用：指定调用动画名称
      2.animation-duration
        作用：指定动画执行一个周期的时长
        取值：以s或ms为单位的数值
      3.animation-timing-function
        作用：指定动画的速度时间曲线函数
	取值：ease,linear,ease-in,ease-out,ease-in-out
      4.animation-delay
        作用：指定动画的延迟时间
	取值：以s或ms为单位的数值
      5.animation-iteration-count
        作用：指定动画的播放次数
	取值：
	  1.具体数字
	  2.infinite 无限次播放
      6.animation-direction
        作用：指定动画的播放方向
	取值：
	    1.normal 默认值，正常0%~100%
	    2.reverse 逆向播放 100%~0%
	    3.alternate 轮流播放
	      奇数次数播放时，正向播放
	      偶数次数播放时，逆向播放
      7.动画简写形式
        animation:name duration timing-function delay iteraton-count direction;
      8.animation-fill-mode
        作用：规定动画在播放前或播放后的状态
	取值：
	  1.none 默认值
	  2.forwards 动画播放完成后，将保持在最后一个帧的状态上
	  3.backwards 动画播放前，延迟时间内，动画保持在第一个帧的状态上
	  4.both 
	    forwards和backwards的结合
      9.animation-paly-state
        作用：指定动画是处于播放还是暂停的状态上
	取值：
	  1.paused 暂停
	  2.running 播放
